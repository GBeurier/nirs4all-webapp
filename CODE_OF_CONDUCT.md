# Code of Conduct

This project follows the same Code of Conduct as the nirs4all library.

Please refer to the [nirs4all Code of Conduct](../nirs4all/CODE_OF_CONDUCT.md) for the full text.

In short: be respectful, inclusive, and constructive. Harassment and discrimination are not tolerated.

## Reporting

Instances of abusive or unacceptable behavior may be reported to the project maintainers.
All complaints will be reviewed and investigated promptly and fairly.
